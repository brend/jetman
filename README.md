## Run Jetman

[Install the Rust tools](https://www.rust-lang.org/tools/install):

``` sh
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

Compile and run from within the project root directory:

``` sh
cargo run
```
